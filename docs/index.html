Demos, lecture slides, and how-to guides for working with AI agents and skills.

For students and colleagues adapting this material for their own teaching. A companion to the Open Science Skills toolkit, maintained by Steven Denney.

Getting started start here

New to this? Setup is a one-time thing. Install Claude Code, add the Open Science Skills, and point it at a project. The walkthrough takes it step by step, with nothing assumed.

Set up Claude Code and the skills →

Install the command-line AI tool that runs the skills, sign in, add the Open Science Skills, and confirm they loaded. About five minutes, no prior setup assumed.

Reference & source-claim checking →

Catch fabricated or malformed citations against your reference list alone. Then, with a small set of your sources on hand, check whether each one actually supports the claim it backs. Runs offline on a short sample article with planted errors.
