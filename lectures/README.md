# Lectures &amp; slides

Slide decks and notes from talks and faculty workshops on using agentic AI in
research and the classroom. Decks are added here as they are given.

Suggested layout, one folder per talk:

```
lectures/
└── YYYY-MM-venue-topic/
    ├── slides.pdf        # the deck (PDF travels best; keep the source too if you like)
    ├── README.md         # one paragraph: where, when, audience, abstract
    └── notes.md          # optional speaker notes or a handout
```

Keep large source files (Keynote, PowerPoint) out of git if they bloat the repo;
export a PDF for distribution and link the source from the per-talk README.
